# DecodeLabs Internship – Data Analytics Project 1

## Project Overview

This project focuses on Data Cleaning and Preparation using Microsoft Excel. The objective was to clean a raw dataset by identifying missing values, checking for duplicates, correcting data formats, and preparing the data for future analysis.

## Objectives

- Identify missing values
- Remove duplicate records
- Correct data formats
- Prepare the dataset for further analysis

## Tools Used

- Microsoft Excel

## Dataset

- Raw Dataset: Dataset for Data Analytics (1).xlsx
- Cleaned Dataset: DecodeLabs_Project1_Cleaned.xlsx

## Data Cleaning Tasks Performed

- Identified missing values in the CouponCode column
- Handled missing CouponCode entries
- Verified there were no duplicate OrderIDs
- Verified and standardized date formats
- Validated dataset consistency

## Results

- Total Records Analyzed: 1200
- Duplicate OrderIDs Found: 0
- Incorrect Date Formats Found: 0
- Dataset successfully cleaned and prepared for analysis

## Outcome

The dataset was successfully cleaned and prepared for future analytics, reporting, and dashboard development.
